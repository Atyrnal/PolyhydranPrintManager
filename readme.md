A project initally developed for the UMass Amherst Physical Computing Makerspace to manage their 3D Printing operations, including user-facing print managment and authorization integrating with existing software such as PrusaSlicer and OrcaSlicer and hardware interfaces (Both open and proprietary) without UX disruption.

Initial demo won 2nd prize overall at HackUMass XIII, a 48 hour hackathon, as a solo project.
